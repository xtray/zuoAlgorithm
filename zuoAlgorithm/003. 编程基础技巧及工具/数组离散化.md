# 数组离散化

---

```java
// 怎么做离散化
// [45,6,89] -> [1,0,2]
public static void change(int[] arr) {
    // [45,6,89] -> [45,6,89]
    int[] copy = Arrays.copyOf(arr, arr.length);
    Arrays.sort(copy); // [6, 45, 89]
    HashMap<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < copy.length; i++) {
        map.put(copy[i], i);
    }
    // 6 ->0
    // 45 -> 1
    // 89 -> 2
    for (int i = 0; i < arr.length; i++) {
        arr[i] = map.get(arr[i]);
    }
}
```
